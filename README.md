#  Tic Tac Toe Game

A classic **Tic Tac Toe** game where two players take turns marking **X** and **O** on a 3x3 grid, aiming to get three in a row horizontally, vertically, or diagonally

---

##  Game Features

- Two-player local gameplay with alternating turns.  
- Automatic win detection for all rows, columns, and diagonals.  
- Draw detection when all cells are filled with no winner.  
- Reset/New Game option to quickly start over.
